# 3D Ray Tracer

This is a basic implementation of a 3D ray tracer in JavaScript.

## Features
- Ray-sphere intersection
- Phong lighting model with ambient, diffuse, and specular components
- Multiple light sources (point lights)

## Installation
1. Clone this repository.
2. Open `index.html` in your browser.

## Usage
- The canvas will render a simple 3D scene with one sphere and two point lights.
